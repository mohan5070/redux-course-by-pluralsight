# redux-course-by-pluralsight
根据pluralsight的《Building Applications in React and Redux in ES6》一步步搭建

## 启动步骤
* 1、npm install
* 2、npm start -s

## 搭建开发环境(Dev environment)
* 1、Babel   //编译jsx
* 2、Webpack //项目构建
* 3、ESLint //代码lint检测
* 4、Mocha //测试
* 5、Express // Web应用框架
* 6、npm Scripts //npm脚本——构建运行命令行

## 介绍
